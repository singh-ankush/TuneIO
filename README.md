🎸 TuneIO

Professional String Instrument Tuner

TuneIO is a modern, browser-based tuner designed for string instruments. It uses your device's microphone and real-time pitch detection to help musicians accurately tune their instruments with a clean, responsive, and professional user interface.

🌐 Live Demo: https://singh-ankush.github.io/TuneIO/

🎸 How to Use

Open the application.

Select your instrument.

Choose a tuning preset.

Select the string you want to tune.

Click Start Tuning and allow microphone access.

Play the string on your instrument.

Watch the tuning meter:

Flat → Tighten the string.

Sharp → Loosen the string.

Perfect → Your string is in tune.

Use Play Note to hear the target pitch.

🔒 Privacy

TuneIO processes audio directly within your browser.

No audio recordings are stored.

No audio is uploaded to any server.

Microphone access is used solely for pitch detection.
